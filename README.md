# Starter pack

This is a starter pack for prototyping or make challenges.

## Use
  
  Clone this repo with git : 
  ```sh
  git clone https://github.com/xmanu91/starter-pack.git
  ```
  
  After this, run the serv with vite :
  ```sh
  yarn dev
  ```
  
 and run sass live compiling :
 ```sh
 yarn sass
 ```
